## Energy Matrix in Chile (2016–2024)

This project analyzes the evolution and diversification of Chile’s electricity generation matrix
using publicly available hourly generation data aggregated to monthly level.

### Key Insights
- Renewable energy shows a sustained growth trend since 2016.
- Conventional generation declines gradually over time.
- Solar and wind technologies drive recent diversification.

### Tech Stack
- PostgreSQL (ETL & data modeling)
- Power BI (Visualization & storytelling)

### Validation
Monthly totals were validated against raw hourly data and official regional statistics.
